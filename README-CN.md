# Inkwell — Typora 主题

一套干净、专注阅读的 Typora 主题，提供亮色（**Inkwell**）和暗色（**Inkwell Dark**）两个版本。

## 设计理念

Inkwell 的核心理念是：主题应该退到幕后，让内容站在台前。它不靠强烈的黑白对比和饱和色来吸引眼球，而是用灰调的层次来建立视觉层级 — 标题偏蓝灰，正文是温润的深灰，代码块用微妙的底色把自己和正文区分开，但不喧宾夺主。

## 功能特性

- **双版本**：白底亮色 + 深蓝灰暗色（`#1a2030`）
- **语法高亮**：完整覆盖 CodeMirror token，15+ 种 token 类型独立配色（关键字、字符串、注释、数字、函数、运算符、类型、装饰器、标签等）
- **代码块**：浅灰底色 + 语言标签头部、行号支持、复制按钮样式
- **表格**：圆角阴影、斑马纹、hover 高亮
- **引用块**：左侧强调边 + 淡底卡片
- **键盘标签**（`<kbd>`）：拟物按键样式
- **YAML Front Matter**：卡片布局，字段名和值语法着色
- **搜索高亮**：匹配项琥珀黄，当前选中项深橙
- **聚焦 & 打字机模式**：非活跃段落淡化至 35% 不透明度，辅助专注写作
- **侧边栏 / 文件管理器**：暗色主题完整的文件树样式（Inkwell Dark）
- **打印优化**：干净的打印样式表，链接自动附加 URL
- **响应式**：小屏幕自动调整间距和字号

## 安装方法

### macOS

1. 打开 Typora → **偏好设置**（`Cmd + ,`）
2. 进入 **外观** → 点击 **打开主题文件夹**
3. 将 `inkwell.css` 和/或 `inkwell-dark.css` 复制到主题文件夹
4. 重启 Typora
5. 在 **主题** 菜单中选择 **Inkwell** 或 **Inkwell Dark**

### Windows

1. 打开 Typora → **偏好设置**（`Ctrl + ,`）
2. 进入 **外观** → 点击 **打开主题文件夹**
3. 将 CSS 文件复制到主题文件夹（路径：`%APPDATA%\Typora\themes`）
4. 重启 Typora
5. 在 **主题** 菜单中选择对应主题

### Linux

1. 将 CSS 文件复制到 `~/.config/Typora/themes/`
2. 重启 Typora，在主题菜单中选择

## 字体方案

- **无衬线**（正文和标题）：系统字体栈 — SF Pro（macOS）、Segoe UI（Windows）、思源黑体 / 苹方 / 微软雅黑（中文）
- **等宽字体**（代码）：JetBrains Mono → Fira Code → SF Mono → Menlo → Consolas
- **基础字号**：15px
- **行高**：正文 1.75，代码块 1.65

## 兼容性

基于 Typora 1.x (macOS) 开发测试。主题使用标准 Typora CSS 选择器和 CodeMirror token 类名，跨平台应可正常使用。语法高亮覆盖 CodeMirror（`.cm-*`）、Prism.js（`.token.*`）、highlight.js（`.hljs-*`）和 Pygments（`.highlight .*`）四套 token 体系。
